The visibility of the button can be controlled via the custom css property ```--ai-search-show-query-offcanvas-button-display```.  
The default being: 

```css
display: var(--ai-search-show-query-offcanvas-button-display, block);
```


To override the default, define somewhere the following responsive rules, for instance:

```css
:root {
    --ai-search-show-query-offcanvas-button-display: block;

    @media (min-width: 768px) {
        --ai-search-show-query-offcanvas-button-display: none;
    }
}
```  

Open "Visible On Mobile" and "Hidden On Desktop" stories for a showcase.
