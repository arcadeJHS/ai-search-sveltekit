To show/hide the search filters, you simply use the ```visible``` attribute.

Add it to render the component:

```html
<stg-ai-search-filters-debug visible></stg-ai-search-filters-debug>
```

Remove it to avoid rendering the component:

```html
<stg-ai-search-filters-debug></stg-ai-search-filters-debug>
```
